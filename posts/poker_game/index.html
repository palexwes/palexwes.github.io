Begin the game by creating a car deck; I did this by setting the card values in a list as well as their types, creating an empty array as well. Utilize nested for loops to create the card. This portion of the code sets the card types and values.

Next create a function that evaluates the value of the card. Use if else statements to investigate the card’s characteristics and correctly setting the value of the card so that they having meaning that can be used in the playing of the blackjack game.

Now, two simple functions will be created that will be for a new card and also removing a card from the deck.

Moving from that into the main drive area of the function which is the lengthiest bit of code that can be broken down into three sections which is created with a while loop that is stopped through a variable first created outside of the loop, but edited in the loop has a exit function by taking in the string ‘EXIT’. If the play again variable doesn’t equal ‘EXIT’ it will continue.

First the deck needed to be created so that the game can be ran through, setting the deck through card creation and removal from the deck, values, and totals created earlier.

After this the hand of the dealer should be created to compare values which is set up through setting values.

The third part of the cod is the creation of the if else statements and along with a while loop to run through the game with the inner while loop will stop when a total goes over the value 21. It uses if else statements through more card creation, removal, and value that was added to the total. Along with more if else statements that entice inputs from the user while checking that the rules of the game are being followed, with the ability for a hit or stay from the player.
